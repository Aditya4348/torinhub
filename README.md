# 🚀 Adit's Digital Universe

A modern, expressive, multi-page portfolio built with React.

This is not just a portfolio website.  
This project represents my identity as a **Gen Z system builder, problem solver, and tech visionary**.

It’s a digital reflection of how I think, build, and design systems.

---

## 🌌 About This Website

This website is designed as:

- 🧠 A system-thinking showcase  
- 🧪 A personal tech playground  
- 🚀 A collection of real-world problem-solving projects  
- ✍ A space for ideas and reflections  

The design is modern, immersive, and personal — not corporate.

Built with:
- React (latest version)
- Tailwind CSS
- Framer Motion (animations)
- React Router (multi-page routing)
- Fully responsive (mobile-first)
- Dark mode by default
- Smooth page transitions
- Interactive UI elements

---
